# PySurf

This is a python web browser that allows you to surf the web freely


## Installation

Install PySurf with pip

```bash
  pip install PyQt5
  pip install PyQtWebEngine
  pip install keyboard
```
    
## Features

- Light/dark mode toggle
- Built in Tetris
- Password autofill
- VPN Support (Coming soon)


## FAQ

#### Where is Tetris

To the left of the themes button

#### Are there any shortcuts?

Yes! Pressing Ctrl+T opens a new tab, Ctrl+W closes the browser, Ctrl+R refreshes the page and pressing the back space key on tetris closes the window
